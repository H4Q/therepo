## Presentation (3 min)

Alexander: Vi är lag H4Q, Alexander och CJ från HiQ

Alexander: Vi har tagit fasta på att våra fantastiska 
värdmyndigheter gett oss så mycket data, men det 
verkliga glappet (avgrunden att överbrygga?) ligger i 
verktygen och metoderna att arbeta med datan, och det 
ligger i att förlänga uppmärksamheten på svensk öppen 
data bortom de här 24 timmarna.

CJ: Vi har lagt minst hälften av vår tid på att prata
med värdarna och våra medtävlare. De har åstadkommit
fantastiska saker, och vi har fått så mycket 
inspiration. vi tror att man funnit bara en bråkdel 
av det potentiella värdet den här helgen

CJ: Därför skapade vi H4Q-communityn på Github. Den har
en elegant framsida via Github Pages och Jekyll och
innehåller de centrala delarna: sandlådan där man
kan laborera och prova idéer med kodexempel 
tillgängliga samt artikel-arkivet där man kan blogga
om de mest intressanta idéerna.

CJ: Github har vänt bakåfram på öppen data, 
författarskap är kryptografisk garanterat och man får 
inte längre skrivrättigheter till någon annans data, 
man erbjuder istället sina ändringar, en extremt 
framgångsrik modell. När till och med Microsoft börjat 
publicera sina artiklar för redigering på github 
tycker vi det indikerar i högsta grad på potentiell 
nytta för oss. Med H4Q-communityn vill vi luckra upp 
stuprören som håller vår öppna data isär och 
otillgängligt för allmänheten. Vi som community och
medborgare ska ta emot stafettpinnen myndigheterna
gett oss.

CJ: Inom ramen för det här har Alexander har byggt 
ett java-ramverk där han gjort kul analyser på data 
från SCB och Kolada. CJ har hackat snabbjobbade 
ipython notebooks med geojson från SGU och vi pekar 
på exempel i python, bonobos snabbafakta och 
Sverker Johanssons superproduktiva wikipedia-bots.

CJ: Vårt projekt kommer leva vidare så länge det finns
hackers som vill titta på datan. Så vi vill uppmana er,
låt inte koden samla damm i byrålådan, donera den till 
oss så visar vi upp den, det blir en kortare 
startsträcka nästa hack och oanade samarbeten kommer 
ske tills dess.
