Explanation

This data has been obtained by first fitting the data from here using a linear regression model There are ten graphs: One for each party. We can then, for each region, get the distance to the fitted line for each party, and form an average value of the distance each region has to the fitted model for each of the ten graphs. We then arrive at the presented graph.

Perhaps unsurprisingly, Kiruna is at the top of the list. Kiruna is a rather wealthy region, yet it is also rather common knowledge that Socialdemokraterna traditionally are rather strong there - while most other regions tend to vote more towards Alliansen when their wealth increases.
